# nworf 


## Getting Started

```
git clone https://github.com/wbudoyo07/nworf.git

yarn install

yarn start

```

### Prerequisites
Install  
```
npm install -g nodemon
MongoDB
NodeJs 

open  nworf/routes/api/twilio.js 
insert your API keu
process.env.ACCOUNT_SID
process.env.AUTH_TOKEN

```
